# Projetsansvice

Shopping App – Project Description
The Shopping App is a responsive, multilingual web application designed to provide users with a simple and intuitive shopping experience. The project includes essential e-commerce features such as product browsing, cart management, shopping lists, user authentication, and language/theme customization. It is built using HTML, CSS, JavaScript, PHP (API), and a MySQL database.

Key Features
1. Multilingual Interface
Full language switching system (English/French)
All buttons, labels, product names, and messages updated dynamically
Uses data-lang attributes + centralized translations in lang.js

3. Responsive UI
Fully optimized for mobile, tablet, and desktop
Adaptive layouts for products, cart, shopping lists, and login pages
Mobile-friendly tables, buttons, and quantity controls

5. Product Browsing
Displays products with names, images, descriptions, and prices
“Add to cart” available in both languages
Supports theme switching and footer positioning logic

7. Shopping Cart
Add, remove, and update item quantities
Real-time item count displayed in header
Persistent cart for logged-in users
Auto-clearing cart for guests (security/UX improvement)

9. User Authentication
Login & registration using a PHP API and PDO for secure DB access
Responsive login/signup pages
Language-adaptive welcome text
After login, user initial appears with a dropdown menu

11. Shopping Lists
Users can create custom shopping lists
Add custom products with names and quantities
Supports editing, deleting, and future list duplication
Fully responsive and translated

13. Theme & UI Enhancements
Light/Dark mode toggle (floating button + hero toggle)
Blurred background image on all pages
Consistent footer across pages
Mobile navigation via hamburger menu
Improved product and cart visuals


Technologies Used
Frontend: HTML5, CSS3, JavaScript (ES6)
Backend: PHP (API architecture)
Database: MySQL
